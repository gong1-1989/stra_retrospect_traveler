# 星溯旅人 (Star Retrospect Traveler)

## 项目简介

**星溯旅人**是一款2.5D箱庭探索策略RPG游戏，主打时空解谜与因果叙事。玩家将扮演一位拥有灵能天赋的旅人，在星际间探索，揭开"大坍缩"灾难的真相。

### 核心特色

- **时间褶皱系统**：多层时间线探索，过去与现在联动解谜
- **星核共鸣系统**：与星核意识对话，获得环境操控能力
- **生态共生系统**：与异星生物建立共生关系，基因融合进化
- **因果叙事分支**：玩家选择影响历史走向，多结局设计
- **动态世界演化**：生态系统随时间动态变化

## 架构设计

### 模块化+插件化混合架构

项目采用三层架构设计：

1. **核心模块层**（模块化）
   - 核心模块：游戏引擎、配置管理
   - 渲染模块：2D/3D渲染、场景管理
   - 音频模块：音乐、音效管理
   - 输入模块：键盘、鼠标、手柄输入
   - 资源模块：资源加载、缓存管理
   - 数据模块：存档、配置、数据管理

2. **游戏模块层**（模块化）
   - 玩家系统：玩家属性、成长、装备
   - 战斗系统：战斗逻辑、AI、伤害计算
   - 三大核心系统：时间褶皱、星核共鸣、生态共生
   - 任务剧情：任务系统、对话系统、剧情管理

3. **插件层**（插件化）
   - 内容插件：新关卡、新角色、新剧情
   - 玩法插件：新游戏模式、新机制
   - UI插件：界面主题、HUD扩展
   - MOD插件：玩家创作的MOD

## 编译指南

### 环境要求

- Qt 6.10.1 或更高版本
- CMake 3.20 或更高版本
- C++17 兼容的编译器（GCC 8+, Clang 10+, MSVC 2019+）
- Qt Creator（推荐）

### 使用Qt Creator编译

1. 打开Qt Creator
2. 选择"文件" → "打开文件或项目"
3. 选择项目根目录下的 `CMakeLists.txt`
4. 选择构建套件（Kit）
5. 点击"配置项目"
6. 点击左下角的运行按钮（或按Ctrl+R）

### 使用命令行编译

```bash
# 创建构建目录
mkdir build
cd build

# 配置CMake
cmake .. -DCMAKE_BUILD_TYPE=Release

# 编译
make -j$(nproc)

# 运行
./bin/StarRetrospectTraveler
```

### CMake选项

| 选项 | 默认值 | 说明 |
|------|--------|------|
| BUILD_TESTS | ON | 构建测试 |
| ENABLE_ASAN | OFF | 启用AddressSanitizer（Debug模式） |

## 开发规范

### 代码规范

- **命名规范**：
  - 类名：大驼峰命名法（PascalCase）
  - 函数名：小驼峰命名法（camelCase）
  - 变量名：小驼峰命名法，成员变量加m_前缀
  - 常量：全大写，下划线分隔
  - 文件名：与类名一致

- **格式规范**：
  - 缩进：4个空格
  - 行宽：不超过120字符
  - 大括号：独占一行

- **注释规范**：
  - 文件头注释：包含文件名、功能、作者、版本
  - 类注释：说明类的用途和主要功能
  - 函数注释：说明参数、返回值、功能

### Git规范

- 分支命名：feature/xxx、bugfix/xxx、hotfix/xxx
- 提交信息：type(scope): message
  - type: feat, fix, docs, style, refactor, test, chore
- 代码审查：所有代码必须经过CR才能合并到主分支

## 核心类说明

### GameEngine（游戏引擎）

游戏的核心类，负责：
- 游戏初始化和关闭
- 主循环管理
- 子系统管理
- 游戏时间和FPS统计

### SceneManager（场景管理器）

负责游戏场景的管理：
- 场景创建和销毁
- 场景切换
- 场景更新和渲染

### ResourceManager（资源管理器）

负责游戏资源的管理：
- 资源加载和缓存
- 资源卸载和垃圾回收
- 支持图片、音频、字体等资源

### InputManager（输入管理器）

负责输入设备的管理：
- 键盘输入处理
- 鼠标输入处理
- 输入状态查询

### AudioManager（音频管理器）

负责游戏音频的管理：
- 背景音乐播放
- 音效播放
- 音量控制

### PluginManager（插件管理器）

负责插件的管理：
- 插件发现和加载
- 插件卸载
- 插件生命周期管理
