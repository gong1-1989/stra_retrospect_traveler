# 《星溯旅人》全套文档 - 使用说明

## 📁 文件说明

本压缩包包含两种格式的文档文件，共10个文件：

### 🌐 HTML格式（推荐直接查看）
- `game_design_doc.html` - 游戏设计文档
- `tech_arch_doc.html` - 技术架构文档
- `dev_guide_doc.html` - 开发指南文档
- `config_resource_doc.html` - 配置资源文档
- `project_ops_doc.html` - 项目运营文档

**使用方法**：直接用浏览器（Chrome、Edge、Safari等）打开即可查看

### 📄 XML格式（用于飞书文档导入）
- `game_design_doc.xml` - 游戏设计文档
- `tech_arch_doc.xml` - 技术架构文档
- `dev_guide_doc.xml` - 开发指南文档
- `config_resource_doc.xml` - 配置资源文档
- `project_ops_doc.xml` - 项目运营文档

**使用方法**：在飞书文档中选择"导入"功能，选择对应XML文件导入

2. 如需编辑文档内容，建议导入飞书文档后在线编辑
3. HTML格式文件仅用于预览查看，不建议直接编辑后导入
