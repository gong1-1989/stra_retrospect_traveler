9.11 配置文件模板

本章节提供项目所需的全套配置文件模板，包括CMake构建配置、JSON数据配置、Qt资源文件等。所有模板均包含完整结构和字段注释，开发者可直接使用或根据需求调整。

9.11.1 CMakeLists.txt 主配置模板

项目主CMake配置文件，包含Qt依赖、C++标准、编译选项、子目录管理等完整配置。

cmake_minimum_required(VERSION 3.20)
project(StarRetrospectTraveler VERSION 1.0.0 LANGUAGES CXX)

# 基础配置
set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)
set(CMAKE_AUTOMOC ON)
set(CMAKE_AUTORCC ON)
set(CMAKE_AUTOUIC ON)

# 编译选项
if(MSVC)
    add_compile_options(/W4 /permissive-)
elseif(MINGW)
    add_compile_options(-Wall -Wextra -Wpedantic)
    set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} -static-libgcc -static-libstdc++")
    set(CMAKE_EXE_LINKER_FLAGS "${CMAKE_EXE_LINKER_FLAGS} -Wl,--large-address-aware")
else()
    add_compile_options(-Wall -Wextra -Wpedantic)
endif()

# Qt依赖
find_package(Qt6 REQUIRED COMPONENTS
    Core Gui Widgets Multimedia Network Svg Test
)

# 源文件配置
set(CORE_SOURCES
    src/core/GameEngine.cpp
    src/core/SceneManager.cpp
    src/core/ResourceManager.cpp
    src/core/InputManager.cpp
    src/core/AudioManager.cpp
    src/core/GameConfig.cpp
    src/core/SaveSystem.cpp
    src/core/PluginManager.cpp
    src/core/EventManager.cpp
)

set(GAME_SOURCES
    src/game/Player.cpp
    src/game/Enemy.cpp
    src/game/TimeSystem.cpp
    src/game/ResonanceSystem.cpp
    src/game/EcosystemSystem.cpp
    src/game/CombatSystem.cpp
    src/game/SkillSystem.cpp
    src/game/AISystem.cpp
    src/game/InventorySystem.cpp
    src/game/QuestSystem.cpp
)

set(RENDER_SOURCES
    src/render/RenderManager.cpp
    src/render/CameraSystem.cpp
    src/render/ParticleSystem.cpp
    src/render/SpriteAnimation.cpp
    src/render/TileMap.cpp
)

set(UI_SOURCES
    src/ui/UIManager.cpp
    src/ui/MainMenu.cpp
    src/ui/SettingsDialog.cpp
    src/ui/InventoryPanel.cpp
    src/ui/DialogBox.cpp
    src/ui/TalentPanel.cpp
    src/ui/StarMapPanel.cpp
)

# UI文件
set(UI_FILES
    src/ui/mainmenu.ui
    src/ui/settings.ui
    src/ui/inventory.ui
    src/ui/dialog.ui
)

# 资源文件
set(RESOURCE_FILES
    config/qrc/images.qrc
    config/qrc/audio.qrc
    config/qrc/game_config.qrc
)

# 可执行文件
add_executable(${PROJECT_NAME}
    main.cpp
    ${CORE_SOURCES}
    ${GAME_SOURCES}
    ${RENDER_SOURCES}
    ${UI_SOURCES}
    ${UI_FILES}
    ${RESOURCE_FILES}
)

# 链接库
target_link_libraries(${PROJECT_NAME} PRIVATE
    Qt6::Core Qt6::Gui Qt6::Widgets
    Qt6::Multimedia Qt6::Network Qt6::Svg
)

# 包含目录
target_include_directories(${PROJECT_NAME} PRIVATE
    ${CMAKE_SOURCE_DIR}/src
    ${CMAKE_SOURCE_DIR}/src/core
    ${CMAKE_SOURCE_DIR}/src/game
    ${CMAKE_SOURCE_DIR}/src/render
    ${CMAKE_SOURCE_DIR}/src/ui
    ${CMAKE_BINARY_DIR}
)

# Windows平台部署
if(WIN32)
    install(CODE "
        execute_process(
            COMMAND ${Qt6_DIR}/../../../bin/windeployqt.exe
                    --no-translations
                    \${CMAKE_INSTALL_PREFIX}/${PROJECT_NAME}.exe
        )
    ")
endif()

9.12 资源清单与规范

本章节详细列出项目所需的所有资源类型、格式规格、尺寸要求、命名规范和目录结构。美术和音频制作人员可按照此规范进行资源制作，确保资源符合项目要求。

9.12.1 图片资源规范

所有图片资源统一采用PNG格式（带透明通道）或JPG格式（背景图等无需透明的图片），特殊图标可使用SVG矢量格式。

  • 角色立绘：PNG格式，1024×2048px，透明背景，300DPI，用于对话界面和角色展示
  • 角色精灵图：PNG格式，单帧128×128px，序列帧横向排列，每帧尺寸统一，透明背景
  • 敌人精灵图：PNG格式，单帧64×64 ~ 256×256px，根据敌人大小调整，Boss尺寸更大
  • 场景背景：JPG/PNG格式，1920×1080px，72DPI，无需透明的用JPG以减小体积
  • UI界面切图：PNG格式，按实际需求尺寸，支持九宫格拉伸，边缘留2px透明边
  • UI图标：PNG/SVG格式，32×32px或64×64px，功能图标、物品图标等，透明背景
  • 特效序列帧：PNG格式，单帧256×256px，技能特效、粒子效果等，序列帧横向排列
  • 瓦片图集：PNG格式，瓦片32×32px或64×64px，瓦片紧密排列无间隙
  • 矢量图标：SVG格式，可缩放，用于需要缩放的UI图标，保持清晰

9.12.2 音频资源规范

所有音频资源统一采用OGG格式（压缩率高，质量好），短音效可使用WAV格式。

  • 背景音乐：OGG格式，44100Hz采样率，128kbps比特率，立体声，循环播放
  • 游戏音效：OGG/WAV格式，44100Hz采样率，16bit，单声道或立体声，时长一般不超过5秒
  • 角色配音：OGG格式，44100Hz采样率，96kbps比特率，单声道，人声清晰
  • 环境音效：OGG格式，44100Hz采样率，立体声，可循环播放
  • UI音效：WAV格式，44100Hz采样率，16bit，单声道，短促清晰

9.12.3 命名规范

所有资源文件统一使用小写字母+下划线命名，前缀标明资源类型，便于识别和管理。

  • 图片资源前缀：角色用char_、敌人用enemy_、背景用bg_、UI用ui_、特效用fx_、物品用item_、瓦片用tile_
  • 音频资源前缀：背景音乐用music_、音效用sfx_、配音用voice_、环境音用amb_
  • 命名示例：char_player_idle.png、enemy_slime.png、bg_village.jpg、ui_button_normal.png、sfx_attack.ogg
  • 版本号：修改后的资源可加版本号后缀，如char_player_v2.png
  • 禁止使用：中文、空格、特殊字符、大写字母（除缩写外）

9.12.4 资源目录结构

所有资源文件按类型分类存放，保持目录结构清晰，便于管理和查找。

9.12.5 字体资源规范

项目使用开源免费字体，确保可商用授权，避免版权问题。

  • 主字体：思源黑体（Source Han Sans），Regular和Bold两种字重
  • 字体格式：OTF格式，支持中文、英文、数字
  • 授权方式：SIL Open Font License，免费商用
  • 使用场景：Regular用于正文，Bold用于标题和强调文字

9.12.6 精灵图规范

所有角色和敌人的动画精灵图需遵循统一规范，确保动画播放正常。

  • 帧排列：序列帧横向排列，从左到右依次播放
  • 帧尺寸：同一角色的所有动画帧尺寸必须一致，对齐中心点
  • 帧率：默认12帧/秒，特殊动画可调整为8帧或16帧
  • 锚点：统一在底部中心点，便于定位
  • 命名：角色名_动作名.png，如char_player_idle.png、char_player_run.png

9.12.7 瓦片地图规范

地图瓦片图集需遵循统一规范，确保瓦片地图渲染正确。

  • 瓦片尺寸：默认32×32px，大型地图可使用64×64px
  • 图集排列：瓦片紧密排列，无间隙，从左到右、从上到下依次排列
  • 图集大小：建议图集宽度不超过2048px，高度不超过2048px
  • 图层划分：地面层、物体层、装饰层、碰撞层，至少4个图层
  • 命名：地图名_tileset.png，如village_tileset.png、forest_tileset.png
